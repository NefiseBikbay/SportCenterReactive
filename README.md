# Sport Center Project With React

A single-page sports center promotional website.

[**Local Preview**](https://nefisebikbay.github.io/SportCenterReactive/)

## Features

- **React** component structure (Header, Hero, Classes, Trainers, Footer, etc.)
- Fast development environment with **Vite**
- Small UI interactions such as scroll behavior and menu open/close (using React Hooks)
- Uses a custom **576px breakpoint** for mobile layout
- Integration of **Google Fonts** and **Bootstrap Icons**
- Semantic **JSX** structure
- Organized folder structure (components, assets, css)

## Technologies Used

### Core
- **React**
- **Vite**
- **JavaScript (ES6+)**
- **CSS3**

### Design & UI
- **Custom CSS**
- **Google Fonts**
- **Bootstrap Icons**
